О проекте

Проект "ИТ-сервисы для «Цифрового университета»" (I курс) реализуется в Московском Политехническом университете с целью повышения эффективности образовательного процесса и административной работы за счёт модернизации цифровых сервисов.

Цели проекта

  • Улучшение пользовательского опыта при работе с личным кабинетом
  • Создание мобильной версии ключевых сервисов университета
  • Разработка системы аналитики для контакт-центра
  • Повышение общей цифровой грамотности пользователей

Основные направления

Личный кабинет

Комплексное тестирование личного кабинета на различных устройствах и браузерах, выявление и документирование проблем, взаимодействие с разработчиками для их устранения.

  • Тестирование кросс-браузерной совместимости
  • Создание системы учёта багов
  • Оптимизация пользовательского интерфейса

Мобильное приложение

Разработка мобильной версии личного кабинета с ключевыми функциями для платформ Android и iOS, включая телефонный справочник университета.

  • Разработка интерфейса для Android
  • Создание дизайна для iOS
  • Реализация телефонного справочника

Визуализация данных

Создание системы мониторинга и аналитики для контакт-центра университета на базе платформы Grafana.

  • Подбор конфигурации сервера
  • Развёртывание Grafana
  • Настройка дашбордов для анализа данных
